# homebrew-tap

Welcome to the Aserto Homebrew Tap

For more about Homebrew, a package manager for for macOS and Linux, see the [Homebrew](https://brew.sh/) website.

## Usage


#### Installation 

To install use: 

```console
$ brew install --cask aserto-dev/tap/topaz
```


#### Update

To update use:

```console
$ brew upgrade topaz
```

### Uninstall

To uninstall use:

```console
$ brew uninstall --cask aserto-dev/tap/topaz
```

## Available brews:

| Name                                             | Installation                                 |
| -------------------------------------------------| ---------------------------------------------|
